id INT PK

expires_at DATETIME

created_at DATETIME
시간표 공유 링크
PK = Primary Key  |  FK = Foreign Key (CASCADE DELETE)  |  nullable = SET NULL on delete
모든 FK → users.id 는 CASCADE DELETE 적용 (유저 삭제 시 관련 데이터 전체 삭제)
exam_schedules.schedule_id → schedules.id 는 SET NULL (수업 삭제 시 시험 기록은 유지)
